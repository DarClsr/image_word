# Client（小程序端）

## 项目描述
面向用户的图文生成小程序，包含主题输入、风格选择、模型选择、生成中状态、结果展示与作品管理。

## 技术栈建议
- 微信小程序原生
- UI 组件：TDesign 小程序 / Vant Weapp（二选一）

## 目录规范
- pages/：页面
- components/：通用组件
- services/：API 请求封装
- utils/：工具函数
- assets/：静态资源

## 代码规范
- 命名：小驼峰（js/ts），文件名用短横线
- 组件：保持单一职责
- 页面：逻辑与视图分离，避免内嵌复杂逻辑

## 环境与配置
- 使用 app.json 管理页面
- 环境变量：在 services/config.js 维护 API BaseUrl

## 开发流程
1. 创建页面与路由
2. 接入 API
3. 完成数据联动与状态管理
4. UI 与交互完善
